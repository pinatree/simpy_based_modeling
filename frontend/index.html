Производственная схема обработки железной руды

Визуализация производственного процесса от добычи железной руды до получения железных слитков

Источник (source)
Хранилище (simple_storage)
Фабрика (fabric)
Потребитель (sink)
Телепорт (teleport)
Поезд (train)

Ресурсы в системе:

Железная руда (iron_rude) Железный слиток (iron_ingot)
Управление: Колесико мыши - масштаб, ЛКМ+перетаскивание - панорамирование
Масштаб: 100%

Управление масштабированием:

  • Масштабирование колесиком мыши: Прокрутите колесико вверх/вниз для увеличения/уменьшения
  • Панорамирование: Зажмите левую кнопку мыши и перемещайте схему
  • Кнопки управления: Используйте кнопки + и - для изменения масштаба
  • Кнопки панорамирования: Используйте стрелки для перемещения по схеме
  • Сброс масштаба: Кнопка ⟲ вернет исходный вид
  • Перетаскивание узлов: Перетаскивайте узлы для изменения расположения схемы
